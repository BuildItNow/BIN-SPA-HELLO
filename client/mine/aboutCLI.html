$ npm install -g bitnow-cli
Install bitnow-cli to node global namespace.
$ bitnow init
Generate a new project from template.
$ bitnow list
List available project templates.
$ bitnow view create
Auto generate view source code from template, a view alwayse has a html file and a javascript file.
$ bitnow view list
List available view templates.
$ bitnow update
Update bin framework to latest version, the bin directory contains all framwork source code.
